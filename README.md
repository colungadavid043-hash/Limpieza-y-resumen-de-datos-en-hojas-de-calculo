
# Limpieza y resumen de datos en hojas de cálculo

Se analizaron los datos de ventas del último trimestre para preparar un informe ejecutivo.Se limpiaron, organizaron y extrajo la información clave para la tomar decisiones estratégicas.


## Objetivos 📊

1. Identificar y documentar problemas en un dataset real.

2. Aplicar técnicas de limpieza sistemáticas y justificadas.

3. Organizar datos siguiendo las mejores prácticas profesionales.

4. Calcular métricas clave relevantes para la toma de decisiones.

5. Crear visualizaciones que comuniquen tus insights (hallazgos) efectivamente.

6. Estructurar un informe claro y viable.
